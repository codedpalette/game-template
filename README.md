# Game template

Godot 4 project template. Based on [Maaack/Godot-Game-Template](https://github.com/Maaack/Godot-Game-Template) with the following additions

- Stricter static typing
- Formatting and linting GDScript with [gdscript-formatter](https://github.com/GDQuest/GDScript-formatter)
- Formatting `.gdshader` files (requires `clang-format` to be installed)
